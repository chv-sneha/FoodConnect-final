# FoodConnect - Frontend Only

A React-based food analysis and nutrition tracking application.

## Setup Instructions

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Run Development Server**
   ```bash
   npm run dev
   ```
   The app will open at `http://localhost:5173`

3. **Build for Production**
   ```bash
   npm run build
   ```

## Firebase Configuration (Optional)

If you want to use authentication features:

1. Create a Firebase project at https://firebase.google.com
2. Get your Firebase config
3. Update `src/lib/firebase.ts` with your credentials

## Features

- Food scanning and analysis
- Nutrition tracking
- Meal planning
- Recipe recommendations
- Health insights

## Tech Stack

- React 18
- TypeScript
- Vite
- Tailwind CSS
- Firebase (optional)
